Penjelasan:
1. Tampilan Awal (Menu)
Saat pertama kali menjalankan program, maka menu awal akan muncul diterminal seperti pada gambar. User dapat memasukkan input angka 1-5 untuk memilih opsi yang tersedia.

2. Create (Menambahkan Mobil ke Datalist)
Jika memilih opsi 1, maka terminal akan meminta user untuk meng-input data mobil baru yang ingin ditambahkan. Disini user dapat melakukan input baru untuk memasukkan ID mobil baru kedalam datalist, Merk, Model, serta Status mobil menggunakan 'string' sedangkan Tahun pembuatan dan Harga Sewa menggunakan 'integer'.

3. Read (Menampilkan List Mobil dari Datalist)
Jika memilih opsi 2, maka user akan melihat list mobil rental yang tersedia di sistem. Fitur ini berfungsi untuk melakukan pengecekan database mobil rental yang tersedia.

4. Update (Mengupdate Datalist Mobil Rental)
Jika user memilih opsi 3, maka user dapat memasukkan input untuk melakukan update pada data mobil yang sudah ada. User dapat melakukan perubahan data pada bagian Merk, Model, Tahun, Harga sewa, dan status mobil.
